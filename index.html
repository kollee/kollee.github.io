Memory Forensics of a Virtualbox VM

Sat, Jun 23, 2018 I use VirtualBox to triage and analyze malware. In contrast to VMWares .vmem file, VirtualBox does not use a separate file to store the current memory. Even if you take a snapshot or pause the VM, the current memory is not saved separately but stores the complete state of the VM in a .sav file. Luckily, Philippe Teuwen describes another way to dump the RAM of a (running) VirtualBox VM by using the VirtualBox debugvm capabilities.
